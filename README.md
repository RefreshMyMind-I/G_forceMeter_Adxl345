# G_forceMeter_Adxl345
This is the G force meter using adxl345 and ssd1306 display, each curve represents 0.2G.You can change Frequency and G force in the code.
## Hardware
| Tested On | Result|
|:---------:|:-----:|
| Arduino Uno | Slow Renderig |
| Raspberry Pi pico| Works fine |
| ESP 32 | Bit faster than pico |
| STM32F401CCU6 | Works fine |
